# Python-FastAPI-Template
Template for building backend using python and FastAPI.

## Tasks

## Environment

- Windows 11
- Python 3.12.8
- Pip 24.3.1

## Tech Stacks

- Python
- FastAPI

## Steps to run program

1. Initialize Project

   ```shell
   poetry init
   ```

2. Install Packages
   ```shell
   poetry install
   ```

3. Write `.env` file
   ```text
   PORT="Port number to host project"
   HOST="Address to host project"
   MODE="Runtime Mode"
   ```

4. Run project
   ```shell
   python main.py
   ```
